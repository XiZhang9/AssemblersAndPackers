# AssemblersAndPackers

This assignment is to practice the use of shared memory by different process.

The user can input a number for assembler process, a number for packer process, a number for balls each box can contain, and a number for the maximnum value in shared memory.
The assembler produces different color balls, then puts on buffer, the packer takes the balls fromm shared buffer and pack to boxes. Finally, display the balls each box contains.
